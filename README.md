# IPL 2025 Data Analysis Dashboard

## 📊 Project Overview
This project analyzes IPL 2025 match data using Power BI. The dashboard provides insights into team performance, toss decisions, and match trends.

## 📁 Dataset
- matches.csv (IPL match data)

## 📈 Visualizations
- Matches Won by Teams
- Toss Decision Distribution (Bat vs Field)
- Matches Played per Month (IPL 2025)
- Impact of Toss on Match Outcome

## 🛠 Tools Used
- Microsoft Power BI

## 📌 Key Insights
- Some teams perform better than others in terms of match wins
- Toss has a moderate impact on match outcomes
- Match distribution varies across months
